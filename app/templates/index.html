Korkeasaari visitor prediction based on weather forecast

The aim of this software is to predict Korkeasaari zoo daily visitor counts based only on weather and date information using machine learning techniques. The weather information is provided by Finnish metheorological institute API and Korkeasaari vistor counts by Helsinki region infoshare.
